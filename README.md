If you're using this module, and need help, you can:

- Tweet to [@senecajs][],
- Ask on the [Gitter][gitter-url].

## Install
To install, simply use npm.

```
npm install seneca-entity
```
## Using Seneca Entity

```
require('seneca')()
  .use('entity')
```

## Test
To run tests, simply use npm:

```
npm run test
```
